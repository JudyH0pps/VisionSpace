# 자율PJT - C102

- WebRTC 사용법
  - npm install 실행
  - npm run devStart 실행 (S는 대문자)
  - 터미널 하나 더 생성해서 (단축키 ctrl+Shift+`) peerjs --port 3001 실행
  - http://localhost:3000/ {원하는 방이름}
  - 크롬에서 새로운 창을 열어서 똑같은 방으로 들어가면 사용자 추가
  - console log를 통해(F12) 누가 나갔고 누가 들어왔는지 확인 가능
